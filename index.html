Getting Started

Need is a powerful Discord bot designed to elevate your server with moderation, utility, and customization features. To begin, invite the bot using the button above and ensure it has the appropriate permissions.

Permissions Required

  • Administrator (Recommended)
  • Manage Roles – For moderation and auto-role features
  • Manage Messages – To delete messages with moderation commands
  • View Channels, Send Messages, etc.

Command Prefix

By default, Need uses / slash commands. No prefix configuration is needed. Simply type / in your server to see available commands.

Features Overview

  • Moderation: Ban, kick, mute, warn, slowmode, etc.
  • Utility: Polls, timers, reminders, embeds
  • Voice Control: Lock/unlock voice channels, auto move, VC mute
  • Leveling: Custom XP, ranks (coming soon)
  • Custom Responses: Add reactions or replies to messages
